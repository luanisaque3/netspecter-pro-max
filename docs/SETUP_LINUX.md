# Setup Guide for Linux and Kali Linux

This document provides a step-by-step installation guide for setting up the project on Linux and Kali Linux systems.

## Prerequisites
1. Ensure you have the following packages installed:
   - `git`
   - `build-essential`
   - `python3`
   - `python3-pip`
   - Any other dependencies specified in the project requirements.

## Installation Steps
1. **Clone the Repository**
   Open your terminal and run the following command:
   ```bash
   git clone https://github.com/luanisaque3/netspecter-pro-max.git
   cd netspecter-pro-max
   ```

2. **Install Dependencies**
   You can install the required Python packages using pip:
   ```bash
   pip3 install -r requirements.txt
   ```

3. **Configure the Application**
   Follow any additional configuration steps mentioned in the repository documentation.

4. **Run the Application**
   To start the application, use the command:
   ```bash
   python3 main.py
   ```

## Notes
- Make sure to replace any hard-coded paths according to your system setup.
- If you encounter any issues, check the permissions or consult the community for support.

## Additional Resources
- For more information, visit the official repository: https://github.com/luanisaque3/netspecter-pro-max
